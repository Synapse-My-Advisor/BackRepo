# M.O-Project Backend

Este é o repositório backend do **M.O-Project**, desenvolvido utilizando **Node.js** e **MySQL**.

## Pré-requisitos

Antes de rodar o projeto, certifique-se de que você tenha instalado:

- [Node.js](https://nodejs.org/en/) (versão 14 ou superior)
- [MySQL](https://www.mysql.com/downloads/) (instância local ou remota)
